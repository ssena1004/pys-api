Personel Yönetim Sistemi API

Kurumsal personel ve görev yönetimi için RESTful API

API Hakkında

Bu API, Personel Yönetim Sistemi'nin backend servislerini sağlar. Aşağıdaki endpoint'ler üzerinden erişilebilir:

Kimlik Doğrulama

POST /api/auth/login - Kullanıcı girişi

Kullanıcı Yönetimi

GET /api/users - Kullanıcı listesi

GET /api/users/me - Mevcut kullanıcı bilgileri

Görev Yönetimi

GET /api/tasks - Görev listesi

POST /api/tasks - Yeni görev oluşturma

GET /api/task-assignments - Görev atamaları
